# 产品文档：Figma AI 工作室 (Figma AI Studio)

## 1. 功能概述
Figma AI 工作室是一个强大的设计辅助工具，集成了多项 AI 驱动的功能，旨在提升设计师的工作效率。目前包含两大核心功能模块：

1. 智能多语言翻译：快速将设计中的文本内容进行多语言转换，支持多层级元素的批量翻译，并保持原始设计的完整性。

2. AI 驱动的 SVG 工具：通过自然语言描述生成 SVG 图形，并支持智能编辑现有 SVG。让设计师能够快速创建和调整矢量图形，激发创意灵感。

这款插件致力于将 AI 能力无缝集成到设计工作流程中，让设计师专注于创意表达，同时提高工作效率。

## 2. 核心功能

### 2.1 翻译能力
- 支持 OpenAI 兼容的 API 接口
- 用户可自定义：
  - API Key
  - 模型名称
  - API 接口地址
- 支持中文到英文的精准翻译

### 2.2 操作方式
- 右键菜单翻译选项
- 快捷键支持：
  - `Ctrl + Option + C`：复制并翻译
  - `Ctrl + Option + V`：粘贴翻译结果

### 2.3 翻译范围
- 支持多层级元素选择：
  - Frame（画板）
  - Group（组）
  - 单个元素
- 翻译内容覆盖：
  - 文本框内容
  - 组件名称
  - 图层名称
  - 自动布局框架名称

### 2.4 用户界面
- 简洁的右键菜单集成
- 翻译进度指示器：
  - 显示当前翻译进度
  - 显示总待翻译数量
- 操作结果即时预览

### 2.5 AI 驱动的 SVG 生成与编辑
#### 2.5.1 文本到 SVG 生成
- 支持通过自然语言描述生成 SVG 图形
- 生成功能特点：
  - 支持详细的图形描述输入
  - 实时预览生成结果
  - 可调整生成参数（如复杂度、风格等）
  - 自动优化 SVG 代码以符合 Figma 规范
  - 支持多种生成模式：
    - LLM 直接生成模式
    - 图像生成模式（可选）
    - DALL-E API 集成（可选）
    - Stable Diffusion 支持（可选）
  - 高级参数设置：
    - LLM 参数配置
    - SVG 复杂度控制
    - 风格定制选项
    - 批量参数应用

#### 2.5.2 SVG 智能编辑
- 基于现有 SVG 的增量修改功能：
  - 选择现有 SVG 元素
  - 通过文本描述指定修改需求
  - AI 理解原始 SVG 结构并智能修改
  - 保留原始 SVG 的关键特征和样式
  - 智能调色功能
  - 风格迁移支持
  - 自动优化功能
- 编辑历史记录：
  - 记录每次修改的描述和结果
  - 支持回退到之前的版本
  - 可比较不同版本的差异
  - 支持选择性回退
  - 版本标记功能
  - 可视化差异对比
  - 代码级比较功能

#### 2.5.3 SVG 管理功能
- SVG 资源库：
  - 保存生成的 SVG 历史
  - 为 SVG 添加标签和描述
  - 支持按类别和标签筛选
  - 本地存储支持
  - 云端同步功能
  - 资源分享功能
  - 智能分类系统
  - 标签组管理
  - 全文搜索功能
  - 高级筛选选项
  - 相似图像查找
- 批量处理：
  - 支持批量生成相似风格的 SVG
  - 统一修改多个 SVG 的样式
  - 模板应用功能
  - 批量参数设置
  - 并行处理支持
  - 批量替换功能
  - 批量变换工具
  - 批量优化功能
  - 样式同步系统

## 3. 使用流程

### 3.1 文本翻译流程
1. 选择需要翻译的设计元素
2. 右键点击或使用快捷键触发翻译
3. 插件自动创建选中内容的副本
4. 在副本中进行翻译处理
5. 实时显示翻译进度
6. 完成后自动显示翻译结果

### 3.2 AI SVG 生成流程
1. 打开 SVG 生成面板
2. 输入详细的图形描述文本
3. 设置生成参数（可选）
4. 点击生成按钮
5. 预览生成的 SVG 效果
6. 确认后将 SVG 插入到当前画板
7. 自动保存到 SVG 资源库

### 3.3 SVG 智能编辑流程
1. 选择需要修改的 SVG 元素
2. 打开编辑面板
3. 输入修改需求描述
4. AI 分析并生成修改预览
5. 确认修改效果
6. 应用更改并更新画板
7. 记录修改历史
